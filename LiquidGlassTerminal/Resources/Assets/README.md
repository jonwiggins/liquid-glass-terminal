# Assets Directory

## App Icon

To create an app icon:
1. Design a 1024x1024px icon featuring the liquid glass aesthetic
2. Use Xcode's Asset Catalog to generate all required sizes
3. Place AppIcon.appiconset in this directory

## Icon Design Guidelines

- **Style**: Liquid glass with translucent layers
- **Colors**: Blue/purple gradients with high saturation
- **Effects**: Blur, vibrancy, depth
- **Shape**: Rounded square (macOS standard)

## Suggested Tool

Use [IconSet](https://apps.apple.com/app/iconset/id1123278952) or create manually in:
- Figma
- Sketch
- Adobe Illustrator
